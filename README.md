# Arduino_clock_project-
## Arduino clock on WS2812B led strip  and DS3231 RTC Module

### Things used for this project :
* Arduino nano V3.0
* DS3231 RTC module
* WS2812B led strip 60 leds/m  2m
* Pushbutton switches 3 PCS
* 10k Ohm resistor
* Power Supply 5V 2A   
* Light dependent resistor
* Cable wires
